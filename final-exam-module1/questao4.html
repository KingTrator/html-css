Questão 4

Sabendo que a contagem das bases hexadecimais vai de 0 a F, assinale as verdadeiras:
01) A base numérica hexadecimal vai de 0 a 9 e depois de A a F, e é particularmente útil para definir cores em CSS devido ao seu poder de compactação superior.
02) A maneira correta de alterar a cor de um parágrafo em CSS é {font-style: red;} ou {color: red;}
04) O código em CSS3: 


    
body{
color: blue;
font-size: 16px;
background-color: #FF0000;
}
Está definindo a fonte dos termos dentro da tag <body> como 16px, a cor das letras em #FF0000 e a cor de fundo como azul.
08) O código U+1F600 representa um caracter unicode (U+) em hexadecimal. Comumente é o código de emojis.
